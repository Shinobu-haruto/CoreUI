# CoreUI Environment

**Core** es un tema completo para Linux, compatible con GTK2, GTK3, GTK4, Cinnamon, GNOME Shell.

## Características

- Paleta única Core (oscuro, verde de acento)
- Variantes Light y Compact
- Compatibilidad con apps GTK2/3/4
- Soporte completo para Cinnamon (panel, applets, menubar)
- Soporte GNOME Shell (panel, actividades, notificaciones)
- Metacity / Muffin legacy compatible

## Instalación

### GTK3 / GTK4

```bash
mkdir -p ~/.themes
cp -r GTK3 ~/.themes/Core
cp -r GTK4 ~/.themes/Core
